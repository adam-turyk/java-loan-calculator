# Loan Calculator (Java Console Application)

This is a Java console-based loan calculator application developed as part of my learning in Computer Science.

## Features
- Calculates total loan repayment based on amount, interest rate, and term length
- Calculates monthly repayment amount
- Stores multiple loan calculations in history
- Allows user to view previous loan results using a menu system
- Includes input validation to prevent invalid values

## Concepts Used
- Methods
- Arrays
- Loops (while and for loops)
- User input using Scanner
- Basic mathematical calculations
- Menu-driven program structure

## Purpose
This project was built to improve my understanding of structured programming in Java, especially working with methods, arrays, and user input handling.

## Future Improvements
- Add file saving for loan history
- Allow variable number of loans instead of fixed array size
- Improve calculation accuracy with compound interest
